# Unijos University Portal Source Code & Bug Exploitation Project

## 🌟 Project Overview
This project aims to obtain, monitor, and exploit the source code of the **University of Jos (Unijos) Portal** (`portal.unijos.edu.ng`). Our primary goals are to:
1.  **Secure Source Code:** Capture frontend and backend code, particularly focusing on database access and the Remita payment gateway.
2.  **Exploit Bugs:** Identify and resolve technical vulnerabilities to enhance system reliability and security.
3.  **Ensure Compliance:** Maintain alignment with legal standards such as the Nigeria Data Protection Regulation (NDPR) and Intellectual Property rights.

## 🤝 Collaboration Model
Since there is no direct access to the university's internal IT team, this project operates as an **External Observer**.
- **We (User & AI):** Act as the digital steward, managing the source code lifecycle.
- **Infrastructure:** Hosting the scraper and repository on **vast.ai** and **GitHub**.
- **Stakeholders:** University of Jos Administration and Students.

## 🛠️ Technical Architecture
- **Hosting Platform:** GitHub (`https://github.com/404pf/unijos`)
- **Server Infrastructure:** vast.ai (Dockerized Scraper & Repository)
- **Payment Gateway:** Remita (Integration for Student Fees)
- **Standards:** NDPR/GDPR Compliance, Open Source Licensing (MIT/GPL)
